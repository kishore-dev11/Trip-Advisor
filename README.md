# Tripadvisor Clone (HTML & CSS Project)

## 🎯 Objective

Replicate the official Tripadvisor website UI using **HTML, CSS, and Flexbox/Grid**.
The goal of this project is to practice layout design, responsiveness, and basic UI components.

---

## 📌 Task Description

### 🧱 1. Set Up the HTML Document

* Created a basic HTML structure
* Divided the page into:

  * Header
  * Main Content
  * Footer

---

### 🧭 2. Build the Header Layout

* Designed a responsive header section
* Included:

  * Logo (left)
  * Search bar (center)
  * Navigation links (right)

---

### 📋 3. Create a Navigation Menu

* Built a horizontal navigation bar using `<ul>` and `<li>`
* Added hover effects using CSS
* Highlighted active menu items

---

### 🔍 4. Design the Search Section

* Added a search input field for destinations
* Included filters such as:

  * Price range
  * Ratings
* Styled using CSS for a clean UI

---

### 🧩 5. Develop the Main Content Area

* Used **Flexbox / CSS Grid** for layout
* Created sections for:

  * Featured destinations
  * Popular hotels
  * Top-rated restaurants

---

### 🃏 6. Create Card Layouts

* Designed reusable cards for listings
* Each card includes:

  * Image
  * Title
  * Rating
  * Short description
* Added hover effects and transitions

---

### 📱 7. Ensure Responsiveness

* Used **Media Queries** for different screen sizes
* Optimized for:

  * Mobile 📱
  * Tablet 📲
  * Desktop 💻

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
